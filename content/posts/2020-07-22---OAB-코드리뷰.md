## React, Next.js 사용

React.js와 React 프레임워크인 Next.js를 사용하였다.

### Next.js 사용이유

라우팅을 편하게 하기 위하여 Next.js를 사용하였다.

Next.js는 `pages`폴더 아래에 js 파일 혹은 디렉토리를 생성하는 그 자체로 routing endpoint가 된다는게 매력적이다.

### 카카오 브라우저

1. autologin 시도

- 실패시 (회원가입이 안되어 있을 때) -> 회원가입 절차 진행
- 성공시 -> info 페이지로 리다이렉트

### 일반 브라우저

- login 페이지로 이동
- 클릭 후

* 카카오 브라우저일 시 -> autologin 후 사용자 정보를 보냄
* 일반 브라우저일 시 -> '/login'으로 보냄.
